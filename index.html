Proyectos

Triaje Start

Información

Triaje Start es una aplicación web. Se creo como herramienta pedagógica para el aprendizaje de un método de clasificación de víctimas utilizado por el personal sanitario y de atención ciudadana para clasificar rápidamente a las víctimas durante un incidente de bajas masivas en función de la gravedad de su lesión.

Desafío

Crear una herramienta interactiva donde lo más importante es que el alumno practique y aprenda con ejemplos cercanos a la realidad, y en segunda instancia, evaluar sus progresos.

Solución

HTML, CSS, Bootstrap, JavaScript, Make Human, Blender.


Calendar app

Información

Calendar app es una agenda donde diferentes usuarios pueden guardar información.

Desafío

Este es un ejercicio realizado para el aprendizaje del stack MERN.
MERN es un conjunto de tecnologías utilizados para el desarrollo web de aplicaciones que consta de MongoDB, React JS, Express JS y Node JS como sus componentes. Se ha utilizado Redux.
Se ha desarrollado el Frontend y un sólido Backend desplegado en Heroku: encriptación de contraseñas, gestión de token (JWT), gestión de usuarios, CRUD de los eventos
También se han hecho las pruebas unitarias e integración con Enzyme y Jest

Solución

React, MongoDB, Express JS Node JS, Mongoose, Redux

Empresas

El encargo de este cliente fue la creación de una aplicación web para utilizarla como herramienta docente.

La finalidad de esta aplicación es que los alumnos adquieran los conocimientos y habilidades necesarias para realizar correctamente un triaje básico en un incidente de multiples víctimas.

Infomed comercializa software de gestión sanitaria.

Me ocupé de extraer la información que el cliente tenía en la base de datos de su ERP antiguo, adaptarla e integrarla en el sistema de gestión propio de Infomed para que conservase la información en el nuevo sistema.

Pastry Factory hizo un cambio de ERP adquiriendo el ERP Tryton.

Los módulos de Tryton están programados en Python. Mi función fue el adapatar Tryton a las necesidades de la empresa programando los módulos y los sistemas de información necesarios.
